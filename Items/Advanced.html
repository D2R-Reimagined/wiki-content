Bat files are simply a list of commands that Windows executes in order through a windows command prompt.  This allows the gamer to simply run the file where it can do the work of replacing the needed files instead of the user.  This is fully transparent and editable through notepad reflecting the open source nature of Reimagined.  This is also flexible, in that gamers that have installed their saved games and/or resurrected into a non-default location are able to edit the .bat file for their own accelerated uses.  The default install directory as included should serve most of the community.

Yellow text is a placeholder for the downloadable .bat file.  Red text for warnings.  Including uninstallers to revert back to default - assume this is a good thing.  We could cleanup the process by removing uninstallers, forcing the gamer to reinstall the mod.  Thoughts?

For all of this to work, the .bat file needs to know the source of all these tweak files to overwrite the default mod files.  Therefore, this collection of optional modifications could be included in a seperate zip file OR could be included with every release in the full Reimagination but simply not activated.  So expanded stash replacement files waiting in the advanced folder, but NOT applied by default.  Goal is to keep this easy for a gamer to enable or disable any tweak below, removing obstacles or user errors.

I am not familiar enough with these modified files to know how often or rarely they are changed in-between reimagined versions by the developers:

Will be best to keep the .bat files on this website only, as in the picture above.  Force gamers to read disclaimers & red font warnings, such as the initial install step for the first time the expanded stash is installed.  Do not want excited gamers to blindly loose their gear without visiting the website.

Typically until now the gamer has to either edit or merge multiple folders, opening the door for user errors.  Allowing the computer to execute the overwrite should make turning on or off a modification extremely easy.

The install bat file simply grabs the one file and replaces the file used inside the real final directory the game uses.

Short description of each of the advanced tweaks.  Click the the link on the left or scroll down for the instructions.

MOST TWEAKS BELOW CAN BE INSTALLED OR UNINSTALLED BY SIMPLY RUNNING THE .BAT FILE.

Expanded Stash

This increases the shared stash tabs from 3 to 7.  Every additional tab is able to hold 2.5M gold as well.  More freedom with more storage.

 

Forced Terror Zones

This forces the following very popular zones to always be terrorized!  While changing your computer system's clock can extend throne of destruction or cow level tz sessions, keeping these popular areas terrorized permanently makes your efforts easier.  

Forced terrorized zones: Countess, Summoner, Nilathak, Andy, Meph, CS, Throne of Destruction, Baal, Cows

 

Potion & scroll removal from most monster drops

With potions and scrolls no longer dropping from most mobs, missing important gear becomes less of an issue.

 

Remove the Splash Charm's dust graphic

By default when using the Splash Charm, a red swoosh effect is present letting the gamer know how large the area for the effect is covering.  You can turn off this graphic, while the damage will continue to hit mobs making it easier to see things while fighting.

 

Expanded Aspect Ratio limit (xx% zoomed out)

This increases the viewable area on-screen, where Blizzard has allowed upto 19:10 ratio to be used in the game.  This allows players to teleport horizontally faster than a typical 16:9 ratio.  Currently the steps listed are for Nvidia video card users only & works on 1080p and 4k monitors too.

 

Much Faster Loading game

This reduces the amount of time it takes for both the game's initial loading and also reduces the loading screens when changing inbetween acts/zones.  This has been reported to remove microstutters & slightly reduce the minimum system requirements (except for the hard drive game storage size requirement).

 

Expanded Stash

This increases the shared stash tabs from 3 to 7.  Every additional tab is able to hold 2.5M gold, so this increases the total amount of gold that can be held too.

Before installing this you must empty your shared stash tabs or risk losing items!  

You can create new character mules to temporarily hold the items & then move them back after the installation.

Please backup your saved game folder too when first installing the Expanded Stash incase you make a mistake

THERE IS NO UNDO FILE FOR THIS EXPANDED STASH.  Backup your saved games before applying.

Expanded Stash Mod Directory Install.bat

Run the file above everytime a new version of Reimagined has been installed.  If you have installed the expanded stash before, then you are done! 

However if this is the very first time you're installing the Expanded Stash, you will need to install the second installer below which will remove/destroy items currently inside your shared stash tabs.  Simply download that file and double click and you're done.  NEVER INSTALL THIS SECOND INSTALLER BELOW AGAIN.

First play Reimagined, confirm that your stash tabs are empty & have no gear or items.  If they do have items, create a level 1 character and move the gear temporarily onto them.

Expanded Aspect Ratio limit (xx% zoomed out)

This increases the viewable area on-screen, where Blizzard has allowed upto 19:10 ratio to be used in the game.  This allows players to teleport horizontally much faster than a typical 16:9 ratio.  Currently the steps listed are for Nvidia video card users only, works on 1080p & 4k monitors too.

When playing on a non 19:10 display, this will introduce black bars on the top and bottom of the screen resulting in a little less visible area.  However, the game will be zooming out to compensate, allowing for more loot to be displayed on screen at once.  Allows you to see monsters further on the edge of the screen & extends how far left and right you are able to teleport making runs go even faster.

Only writing the steps for Nvidia Graphics card users for now.  Will update for AMD users this summer.  Note that Windows needs to be operating at this aspect ratio while playing the game in fullscreen to be successful.  So limited visibility while alt+tabbing will remain until you quit the game and change back to your typical desktop resolution.  Suggest trying this at least once, especially if playing on a larger display.
