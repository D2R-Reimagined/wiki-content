Gem Storage

You can create a stack of each quality of gem (up to 30). Combine 3 of the same quality and type of gem to transform them into a stack. Then in your inventory drag and drop the same type of stacks together, and you will add gems to the stack!

What is the Gem Bag?

The gem bag is a custom item that can be used to store several gems of all kinds in it at the same time.

Acquiring the Gem Bag

You can create the Gem Bag by combining each chipped gem in the unstacked version together. You will need to use one of each gem as seen in the picture to your right. DO NOT STACK!

 

 

How it Works

The gem bag uses a "Credit" System where the amount of credits equal the number of chipped gems it would take to get that item.

So here is a breakdown of the credits for each gem:

  • 1 = Chipped Gem
  • 2 = Flawed Gem
  • 4 = Gem
  • 8 = Flawless Gem
  • 16 = Perfect Gem

Storing Gems

Simply cube the gem you want to store with the gem bag in the cube, and you will add the credit count to the gem bag. You can store up to FOUR of the same TYPE and QUALITY of gem at the same time. For example: cube 1-4 chipped gems with the bag and they all get put into it.

Retrieving Gems

First, you must obtain a grabber tool. Each grabber tool corresponds to the version of the gem. So if you want to grab a "Perfect Topaz," you would get the "Perfect Topaz Grabber."

Grabber tools can be obtained by cubing any chipped gem in the unstacked version alone in the cube. This will turn into a grabber for that gem.

Cube the grabber tool with the gem bag together, and it will extract the gem from the gem bag and use the credits that are in the gem bag. If you do not have enough credits in the gem bag for the item it will not work.

Cube a grabber by itself to cycle through the available grabber tiers to create normal, flawed, perfect, etc.
