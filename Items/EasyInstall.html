Diablo 2 Reimagined - Easy Install

 

 

This Easy Install setup only works if your game is installed in the default location below & should work for almost everyone:

C:\Program Files (x86)\Diablo II Resurrected

 

If you are unsure where your game is installed, download and double click this .bat & it will lookup the registry key displaying the games install location.  Below is the default location displayed from the bat file.

 

 

If your game is installed in a different location then instead pick from one of these other options

C:\Diablo II Resurrected

D:\Diablo II Resurrected

E:\Diablo II Resurrected

F:\Diablo II Resurrected

 

 

1. Open the file "1 FIRST RUN ME"

 

2. Now download the reimagined mod off Nexus & save it to:

C:\Program Files (x86)\Diablo II Resurrected\Download Reimagined Updates Here\

 

3. Open the "3 REINSTALLS REIMAGINED MOD WITH NEW VERSION"

 

 

Now the mod is installed!  

 

Easy!

 

 

 

Incase you are curious what happened.  Step one deleted previously saved .7z mod downloads in the step two location as we only want to install the fresh new one you were about to download.  Step two was you downloading the compressed .7z file containing the mod.  Step three the freeware commandline version of 7zip unpackaged the mod & overwrote the previous version.  Remember your saved games are safely stored in an entirely different section.

 

 

 

Bat files are fully transparent & editable by right clicking and choosing edit.  Feel free to change the path to better suit your needs.
