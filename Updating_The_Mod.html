Mod Install Location*: <drive>\Program Files (x86)\Diablo II Resurrected\mods

Character Save File Location*: <drive>\Users\<your user>\Saved Games\Diablo II Resurrected\mods 

*Your directory structure may differ depending on where you chose to install the game and your save files.

 

Version Compatibility

Major patch versions (e.g. 1.X → 2.X) often have changes that may make character saves incompatible. 

The description on Nexus Mods and associated patch notes will tell you when save-breaking changes are introduced.

We strongly suggest making a backup of your Mod folder and Character Save folders before updating the mod, even between minor patches.

 

Updating from 1.x to 2.x

  1. Download the zip from Nexus Mods.
  2. Extract the files into the mods directory of your D2R install. This should create a new folder called Reimagined.
  3. Copy your 1.X shortcut - rename it appropriately so you know which shortcut is which.
  4. Open the properties of your new shortcut. Change -mod Merged -txt → -mod Reimagined -txt
  5. Launch the game using the new shortcut.

 

Updating within the same major version (1.x -> 1.y)

When updating to the latest version of D2R Reimagined from a previous version:

  1. Download the Latest Version from Nexus Mods.
  2. Make a backup of your Merged folder in your mods directory.
  3. Delete the Merged folder in your mods directory.
  4. Extract the new version into the mods directory.


Why do we delete the entire folder?

Sometimes the mod will delete files and if you simply extract over without deleting those files will still exist and could cause issues.

Will my saves be okay?

Unless explicitly expressed in the patch notes that a new save is required your saves are safe.
