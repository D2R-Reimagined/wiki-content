Updating the Mod

 

Updating from the same major version (1.x -> 1.y)

When updating to the latest version of D2R Reimagined from a Previous Version here are the steps to follow.

  1. Download the Latest Version from Nexus Mods
  2. Delete the `Merged` folder in your `mods` directory
  3. Extract the new version in the root of your directory


Why do we delete the entire folder?

Sometimes the mod will delete files and if you simply extract over without deleting those files will still exist and could cause issues.

Will my saves be okay?

Unless explicitly expressed in the patch notes that a new save is required your saves are completely safe. They are stored in "My Saved Games" and not in your D2R Install directory.


 


 

Updating from 1.x to 2.x

  1. Download the file where available
  2. Extract the files in the root directory of your d2r install
  3. Copy your shortcut you have created to launch the modded game
  4. Open the properties of your copied shortcut. Change the section that has `-mod Merged -txt` to `-mod Reimagined -txt`

Now launch the game using your copied shortcut.
