Gems

Gem qualities (e.g. chipped, flawed, normal, flawless, perfect) have been removed.

All gems drop from Act 2, Arcane Sanctuary onwards and require level 18 to use.

Gem Bag

All new characters start with a Gem Bag in their inventory, and replacements can be purchased from vendors should you somehow misplace it. 

The Gem Bag is used to store large numbers of gems as a universal currency. 

Gems can be retrieved using ‘grabbers’ (also purchased from vendors).

If a crafting recipe creates a new item and requires gems you can use the Gem Bag as the Gem reagent (as long as you have enough gems stored), e.g. : REROLL CHARM - Gem Bag (3 Gems) + Magic Grand Charm = New Magic Grand Charm (100% plvl)

If a recipe calls for gems to change an existing item then gems need to be used instead of the bag, e.g.:

ADD ITEM STAT - Amulet & 9 Emeralds = +1 to Druid Skills
