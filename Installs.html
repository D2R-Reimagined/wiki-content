Installation Guides

Below is a compilation of useful information and guides.

File Locations

D2R Mod Files: C:\Program Files (x86)\Diablo II Resurrected\mods\Reimagined

Saved Game Files: C:\Users\%USERPROFILE%\Saved Games\Diablo II Resurrected\mods\Reimagined

Your install and save locations may vary depending on your setup!

We aren't IT support, find your install and saved game directories if it differs from the above (you may have installed D2R on a different drive; you use D2RMM, etc.)

Making Backups (Manual Steps)

In File Explorer, navigate to your D2R install and Saved Games directories.

Right click the mod folder and Reimagined folder, select ‘compress to…’ and choose ZIP file. Rename the ZIP file to something memorable, and move it somewhere easy to remember (e.g. your desktop).

Remember to periodically create new ZIP folders so that you don't lose progress in case of crashes, exceeding your save file data limit, etc..

Download the Reimagined Desktop Launcher

Installing & Updating Reimagined

These steps assume you have already downloaded and unpacked the Reimagined Desktop Launcher (see above)

Your Save Game files are unaffected by these steps

Follow these steps to install and/or update Reimagined to the latest version:

1. Download the latest version of Reimagined from Nexus Mods - this can take a few minutes.

2. Move the downloaded .7z file from your Downloads folder to the Reimagined Desktop Launcher folder.

  • Recommended: delete any previous .7z files from this folder (if any).

3. Double-click ‘Install Reimagined (Install-Update)’.

  • If you see a warning press ‘More Info’ and click ‘Run Anyway’.
  • You should see a black command window open - let the process finish.
    • If you see an error, refer to the troubleshooting section below.

4. The mod is now installed/updated and ready to play!

If you wish to install Advanced Tweaks, read on below.
 

Play & Backup

To play the mod, double-click either of the ‘Play Reimagined’ files. This replaces the shortcut way of launching the mod!

  • ‘Play Reimagined (Daily Backup)’ - Launches the game and creates a backup of your Reimagined Saved Game files with today's date. If a backup exists from today, it will overwrite it.
  • ‘Play Reimagined (Session Backup)’ - Launches the game and creates a backup of your Reimagined Saved Game files each time you launch the game, including the date/time.

Backup files are located under your Saved Games folder, in a folder called Reimagined Automatic Saved Game Backups - see File Locations above.
 

Advanced Tweaks

This section assumes you have run ‘Install Reimagined (Install-Update)’ first (see above)

In addition to installing and updating the mod, you can use the RDL to install pre-packaged changes.

Navigate to the Advanced folder - here you should see a number of files.

To install changes, simply double-click the relevant file(s).

To uninstall changes simply re-run ‘Install Reimagined (Install-Update)’.

As mentioned above, whenever you run ‘Install Reimagined (Install-Update)’ any manual or advanced tweaks changes will need to be redone.

If anything isn't working, refer to the Installation Troubleshooting section below.
 

Expanded Shared Stash Tabs

Description: Adds 4 additional Shared Stash tabs.

Installed this change before? Re-run ‘Expanded Stash Mod Directory Install'.

First time installing? 

Clear all items from your Shared Stash tabs - move them to personal stash, inventory, or to a mule.

Run: 

  1. Expanded Stash Mod Directory Install
  2. Expanded Stash Saved Game Install (this overwrites your existing Shared Stash file)

Run: ‘Forced Terror Zones Install’

Areas terrorized:

  • Abandoned Tower (Countess)
  • Catacombs (Andariel)
  • Arcane Sanctuary (Summoner)
  • Durance of Hate (Mephisto)
  • Chaos Sanctuary (Diablo)
  • Nihlathak's Temple (Nihlathak)
  • Throne of Destruction
  • Worldstone Chamber (Baal)
  • Moo Moo Farm (Cows)
     

Remove Splash Charm Dust Effect

Description: Removes the dust particle effect when using the Splash Charm (Collin's Charm).

Run: ‘Splash Charm Effect Graphic Removal Install’


Two Skill Points / Level

Description: Characters gain 2 skill points per level instead of 1.

Run: ‘Two Skills Per Level Install’

  • This change does not work retroactively.
  • Skill points already earned cannot be modified.
  • If you uninstall this change characters will return to earning 1 skill point per level.
  • Using a Token of Absolution to reset your character will refund all skill points earned up to that point (whether 1 or 2 points).
     

CASC Fastloading

Description: Unpacks all game files - reduces load times, increases performance and FPS, client response times are better, etc.

NOTE: Requires an extra 41GB of disk space to unpack

Run: 

  1. Step 1. CASC Fastloading (takes 5-10 minutes to complete)
  2. Step 2. CASC Fastloading
  3. Install Reimagined (Install-Update)
    • Note: this will uninstall other advanced tweaks, re-install them as desired afterwards

 

Installation Troubleshooting

Below is a list of possible failure points you may encounter during RDL installation:


Error: Cannot find 7zr.exe / Path Too Long
  • Problem: You didn't extract to the desktop.
  • Resolution: Double-check where you extracted to, move the folder to your desktop if necessary.

Error: Reimagined isn't loading!

Problem: Reimagined 7z file not in RDL folder.

Resolution: Ensure you have downloaded the latest Reimagined 7z file from Nexus and moved it to the Reimagined Desktop Launcher folder.

  • Re-run Install Reimagined (Install-Update) to load the mod, and try again


Error: Advanced Tweaks aren't working!

Problem: Advanced folder isn't in the Reimagined mod folder. You likely didn't run Install Reimagined first.

Resolution: If your advanced folder is missing, here are a few things to try:

1. Run Install Reimagined (Install-Update).

  • Once done, try re-applying your advanced tweak changes

2. Redownload RDL, repeat step 1 above.

3. If still not working, try manually downloading the advanced folder here. Extract it and move the folder to your \Diablo II Resurrected\mods\ folder. Repeat step 1 above.


D2RMM

D2RMM on Nexus Mods

Using a Mod Manager allows you to easily install a variety of mods together (assuming they play nice) without having to know anything about modding.

Your mod files will be now be located in D2RMM\mods\ instead of Diablo II Resurrected\mods\.

Any changes you make to Reimagined files needs to be done in the D2RMM mods directory.

If using the RDL (above), you will need to copy the file changes over from Diablo II Resurrected\mods\ to D2RMM\mods\.

Loot Filters

This guide assumes you have already downloaded and installed D2RMM (see above)

If you find Quilted Armor with no graphics, leave them on the ground. Do not attempt to identify or sell them.

Reimagined doesn't yet include a built-in Loot Filter.

One of the most popular reasons to use a Mod Manager is to install a 3rd-party Loot Filter to help reduce clutter.

Below we'll take you through a step-by-step guide on how to install and configure a loot filter. We recommend Pseudo Loot Filter.
 

File Steps

1. Open your D2RMM folder and navigate to the mods directory

  • You should see a text file called 'PUT YOUR MODS HERE' - leave this window open, we'll use it later

2. Open a new File Explorer window and navigate to your Reimagined mod folder

3. Copy the Reimagined.mpq folder (this should contain Data and ModInfo.json)

4. Paste the Reimagined.mpq folder into the D2RMM/mods folder from step 2

  • Feel free to rename Reimagined.mpq to something that makes more sense to you (e.g. Reimagined)

D2RMM Steps

5. Open D2RMM as Administrator. Once open, click the three dots in the bottom-right corner and select 'Refresh Mod List'

6. Reimagined.mpq should appear in the mod list (or whatever you renamed the folder to)

The Reimagined folder must always be first in the list of mods. Click and drag the line to the top if it isn't.

7. Navigate to the Settings tab (at the top)

  • 7a. Click on General
    • Change 'Output Mod Name' to 'Reimagined'
    • Toggle 'Automatically install mods before running the game'
  • 7b. Click on Game Launcher
    • Under 'Extra Game Args', paste the following: -mod Reimagined -txt
    • Feel free to add any other args you want here

8. Navigate back to the Mods tab

9. Click Install Mods


Save File Steps

10. In File Explorer, navigate to your Saved Games directory

11. Check the Reimagined folder - if you see your save files, you're good to go! 

  • If the folder is empty, copy the contents from the backup folder you created earlier, and paste the contents in the empty Reimagined folder.
     

Final step

12. Hit 'Run D2R' in D2RMM and have fun!


FAQ:

My mod isn't showing up in the mod list on D2RMM?

  1. Either you didn't put the Reimagined.mpq folder in D2RMM/mods or it's in a sub-folder called Reimagined
  2. Run D2RMM as administrator
  3. You may need to follow step 8 to update the Output Mod Name
