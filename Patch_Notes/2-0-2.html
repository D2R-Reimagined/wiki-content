General Changes

  • Javelins should now properly only allow 5 upgrades
  • Added elemental tags to non-damage skills so that they will now benefit from the crafted +Elemental Skills crafts
  • Fixed some NM TZ drop rates for various super uniques
  • Summoner key drop rates increased to match the other key holders
  • New Lieutenant Super Uniques (e.g. Raven Talon) now have proper loot tables
  • Fixed missing languages in item-names for several languages. Evil Force should display properly now on new potions.
  • Winged Helm now properly shows [X] to indicate it is an Exceptional base

Horadric Cube

  • Crafted Grand Charm recipe w/ Pul damage reduced from +20-30 to +10-18 to be more in line with other affixes for the cost
  • Fixed some recipes that had the incorrect gem, rune, and jewel requirements
  • Fixed some recipes that had tal/ort/amn rune numbers mixed up
  • Fixed some item upgrade for amulet recipes that did not have the proper gem required
  • Fixed some item upgrade for amulet recipes that were not requiring orb of infusion as they should
  • Fixed new quiver recipes to require rare jewel instead of Fal Rune as intended.
  • Fixed the helm craft recipes all defaulting to +Fire Skills, now proper displays the +cold/lightning skill levels added
  • Fixed item upgrade for helm all skills recipe syntax to actually spawn properly
  • Fixed Weapon Upgrade recipe for %ED/+AR to now correctly use a Cham rune instead of Jah

Monsters

  • Nerfed Talic's Static on death from 90% HP to 75% HP
  • Raven Talon and Raven Eye will now only spawn once per game

Items

Note: Changes are not retroactive and will not impact existing items

Uniques

  • Hill Giant's Bludgeon
    • +Max damage decreased from +(300-350) to +(200-250)
    • +Min damage decreased from +(120-150) to +(90-120)
  • Wave Whipper
    • Removed useless Increased Stamina oskill
  • Ecstacy of Ishtar
    • %MaxMana and %MaxLife reduced from 25% -> 10%
    • Flat PDR reduced from 25-30 -> 18-25
    • Flat MDR reduced from 25-30 -> 10-15
    • Required level increased from 71 -> 80
    • ilvl increased from 65 -> 86 (hell drop only)

Affixes

  • Ring's can no longer roll the "of Life Everlasting" suffix (highest flat reduction tier)
  • Poison Dagger charges should now only roll on daggers
  • Tectonic Slam charges should now only roll on slam weapons

Charms

  • Mana after kill from 3/6/10 -> 1/2/3
  • Life per hit from 3/6/10 -> 1/2/3
  • Disabled magic resist rolls
    • This affix on charms is being discussed internally as many monster skills are balanced around 0 mag-res, so the affix shouldn't be so easily obtainable
  • Disabled Hulking affix on grand charms (added flat normal damage)
    • This is redundant with the Red/Sanguinary/Bloody affix on charms which have more reasonable values.
  • Level requirement on crafted sunders reduced to hopefully reduce the amount of lvl99 required outputs from crafting sunders
