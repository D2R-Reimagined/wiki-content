Installation Guides

Below is a compilation of useful information and guides.

Before getting started, take a second to make a backup of your files (this is always good practice before making changes)!

File Locations

Mod Files: C:\Program Files (x86)\Diablo II Resurrected\mods\Reimagined

Save Files: C:\Users\<YOURNAME>\Saved Games\Diablo II Resurrected\mods\Reimagined

Your install and save locations may vary depending on your setup!

Making Backups

In File Explorer, navigate to your your D2R install and Saved Games directories.

Right click the directory, select ‘compress to…’ and choose ZIP file. Rename the ZIP file to something memorable, and move it somewhere easy to remember (e.g. your desktop).

Remember to periodically create new ZIP folders so that you don't lose progress in case of crashes, exceeding your save file data limit, etc..

Loot Filters

This guide assumes you have already downloaded and installed D2RMM (see above)

If you find Quilted Armor with no graphics, leave them on the ground. Do not attempt to identify or sell them.

Reimagined doesn't yet include a built-in Loot Filter.

One of the most popular reasons to use a Mod Manager is to install a 3rd-party Loot Filter to help reduce clutter.

Below we'll take you through a step-by-step guide on how to install and configure a loot filter. We recommend Pseudo Loot Filter.
 

File Steps

1. Open your D2RMM folder and navigate to the mods directory

  • You should see a text file called 'PUT YOUR MODS HERE' - leave this window open, we'll use it later

2. Open a new File Explorer window and navigate to your Reimagined mod folder

3. Copy the Reimagined.mpq folder (this should contain Data and ModInfo.json)

4. Paste the Reimagined.mpq folder into the D2RMM/mods folder from step 2

  • Feel free to rename Reimagined.mpq to something that makes more sense to you (e.g. Reimagined)

D2RMM Steps

5. Open D2RMM as Administrator. Once open, click the three dots in the bottom-right corner and select 'Refresh Mod List'

6. Reimagined.mpq should appear in the mod list (or whatever you renamed the folder to)

The Reimagined folder must always be first in the list of mods. Click and drag the line to the top if it isn't.

7. Navigate to the Settings tab (at the top)

  • 7a. Click on General
    • Change 'Output Mod Name' to 'Reimagined'
    • Toggle 'Automatically install mods before running the game'
  • 7b. Click on Game Launcher
    • Under 'Extra Game Args', paste the following: -mod Reimagined -txt
    • Feel free to add any other args you want here

8. Navigate back to the Mods tab

9. Click Install Mods


Save File Steps

10. In File Explorer, navigate to your Saved Games directory

11. Check the Reimagined folder - if you see your save files, you're good to go! 

  • If the folder is empty, copy the contents from the backup folder you created earlier, and paste the contents in the empty Reimagined folder.
     

Final step

12. Hit 'Run D2R' in D2RMM and have fun!


FAQ:

My mod isn't showing up in the mod list on D2RMM?

  1. Either you didn't put the Reimagined.mpq folder in D2RMM/mods or it's in a sub-folder called Reimagined
  2. Run D2RMM as administrator
  3. You may need to follow step 8 to update the Output Mod Name

DM on discord if you need further instructions/help

Can easily Install Reimagined

First you must download the reimagined mod off Nexus and save the .7z into the “Reimagined Desktop Launcher” folder on the desktop.  Simply double click the “Install Reimagined (Install-Update)” & the mod will get installed!  Works if you've never installed the mod before, if you want to re-install the mod or if you're looking to update to a new version of the mod.

 

Automatic Saved Game Backup

Running this both launches Reimagined & also copies your Resurrected saved game directory into a new zip file at the same time.  So by playing the game, you've just created a saved game backup.  These new saved game backup zips are stored in a new location in: \Saved Games\Reimagined Automatic Saved Game Backups\

Play Reimagined (Daily Backup): Saves only one zip backup a day, overwriting that zip each time you play until the next day where a new zip will be made.

Play Reimagined (Session Backup): Saves a new zip backup everytime you launch the game.  However, if you run the game 200 times you'll have 200 zip files to deal with using this option.

Reimagined Desktop Launcher - Advanced

These are optional tweaks that can be installed very easily by simply double clicking the ones that are of interest to you.  They are located inside the “Advanced” folder within the “Reimagined Desktop Launcher” folder on your desktop.  If you simply reinstall the reimagined mod, then all the tweaks in the advanced folder reset back to being turned off again.  This allows you to easily turn on or off any tweak.

 

Expanded Stash

This increases the shared stash tabs from 3 to 7.  Every additional tab is able to hold 2.5M gold as well allowing for an additional 10 million more gold to be stored.  More freedom with more storage.  This is Delegus's expanded stash & may need to be updated with a fresh "advanced tweaks.zip” as it'll be included.

 

Forced Terror Zones

This forces the following very popular zones to always be terrorized!  While changing your computer system's clock can extend throne of destruction or cow level tz sessions, instead this will keep these popular areas terrorized permanently makes your life easier.  This ensures key farming areas are always terrorized too!

Forced terrorized zones: Countess, Summoner, Nilathak, Andy, Meph, CS, Throne of Destruction, Baal, Cows

 

Remove the Splash Charm's effect graphic

By default when using the Splash Charm, a red swoosh effect is present letting the gamer know how large the area for the effect is covering.  You can turn off this graphic, while the damage will continue to hit mobs making it easier to see things while fighting.

 

Two Skills Per Level

By default characters earn 1 skill per level.  This modification changes it so characters earn 2 skills per level.  This will only work for characters that have earned levels while it was turned on.  If a character has gained levels while this was turned on, then that character gets to keep those levels even after turning this off.  The extra skills and attributes remain even when resetting your skills.
