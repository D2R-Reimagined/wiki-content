Reimagined Installer

 

This Reimagined Installer is very simple to use.  You download the latest version of Reimagined off of Nexus & save it into this Reimagined Installer folder.  Then you run the bat file and now your mod is installed.  If you already have reimagined installed, this will overwrite your install & update the mod.  No 3rd party software will need to be installed, but a freeware commandline tool will need to be downloaded once.  This makes updating your Reimagined fast & easy!  Instructions are included in the readme file inside the zip.

How the Reimagined Installer works

 

There is a bat file along with a readme instruction included inside the zip.  A freeware command line version of 7zip will need to be downloaded as mentioned inside the readme.  This version of 7zip does not install itself onto your system.  This installer should never needs to be redownloaded.  It's purpose is to simply make uncompressing future reimagined downloads even easier.  No longer need any installed software to uncompress the Reimagined mod.

 

EASY INSTRUCTIONS

1. Navigate to your \Diablo II Resurrected\

2. Unzip the file dragging the Mods folder into an empty space merging it with the existing Mods folder.  Now you'll have a new folder in: 

\Diablo II Resurrected\Mods\Reimagined Installer  

3. Download the latest version of Reimagined off Nexus & save that large .7z file into that new folder listed above.

4. Finally doubleclick the bat file & the reimagined mod will be uncompressed and overwrite your old mod.  Your saved games are located in a different location and should remain safe when updating the mod.  This will work if it's your first time installing Reimagined or if you simply want to update your existing installation with a new version.

 

Now Reimagined is installed!

 

Bat files are fully transparent & editable by right clicking and choosing edit with notepad.  A bat file is simply Microsoft Windows Command Prompt executing commands one after another.
