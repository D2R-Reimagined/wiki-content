Characters

- Skills cap out at 25 hard points, up from 20.

- Many skills and their synergies have been changed - you can see a summary of these changes in-game by pressing Escape and looking at the Class Balancing section.

- You can respec at any time by cubing a TP Scroll and an ID Scroll in the cube to give you a token of absolution.


 

Items

- Check the website pages to find items. - Note: Some Set Names haven't been added fully. Find the items you want by doing a page search (Ctrl + F).

- Stacked Items: Runes drop as 'stacks' (the + sign in the top-left is a visual indicator), which can store up to 100 in a stack. Cube a stack to create a rune stack with qty=1. Cube a stack of qty=1 to revert it to a non-stacked state to use it as normal.

- Retrieve socketed items: Item + TP Scroll, returns all socketed gems/jewels/runes. - Pro Tip: can be used to endlessly re-roll Runewords!

- Re-roll Items into White Bases: Item + Hel Rune + El Rune, re-rolls the item as a white base with new affixes

- Item Augmentation: Check the Add Item Stat Recipes page

- Crafting: Check the Item Crafting about how to craft and alter various items

- Essence drops have been replaced by Socketing orbs.

- Many new Runewords have been added - while you level keep checking the runeword page on the website for new and exciting options for your build!

- Jewels can be used to 'socket-punch' items when you want a specific number of sockets (as long as the item supports it). Item Rarity = Jewel Rarity required to punch sockets with. * Note that number of open sockets is dependent on the ilvl of the item. Check https://maxroll.gg/d2/items/sockets under 'Item Sockets By ilvl' for details.

 

Quality of Life

- Cube a rune with ID Tomb to cube a rune down. 
- Cube Two of the same rune with a TP Tomb to cube a rune up.

- Runes and Gems can be stacked for better inventory management - cube three single runes or gems together to form a new stack.

- You can store all your Gems in a gem bag, which is created by cubing all 7 gems together. 

-Gem Grabbers are used to retrieve gems from the bag. Gem Grabbers can be purchased from most vendors. You can cycle Gem Grabbers by cubing them by themselves to choose which gem type you would like to extract.

- Keys can be stored in a Keychain, and extracted with the appropriate key grabber. Grabbers can be found at many vendors.

- There is a charm (Collin's Might) that allows your melee attacks to have splash damage that drops from Corpsefire in the Den of Evil on normal (once per character). It can be upgraded a number of times via cube recipes to deal more damage and have less penalties.

 

Due to character file size limitations in place by D2R it is recommended to keep STACKED ITEMS, KEYCHAINS, and GEM BAGS in a SHARED STASH TAB to prevent items from disappearing.
