Keychain

The keychain is for holding multiple copies of Infernal Keys (Key of Hate, Destruction, and Terror).

Acquiring

You can purchase a keychain from any vendor.

How it Works

The keychain uses a "credit" system. Each key that is added to the keychain adds one equivalent credit. Key credits are displayed separately.

Storing Keys

Transmute an infernal key with the keychain in the Horadric Cube.

Retrieving Keys

Purchase a key grabber from any vendor - there is a key grabber for each Infernal Key.

Transmuting the keychain with a grabber extracts a key from the keychain and subtracts a credit from the keychain.
