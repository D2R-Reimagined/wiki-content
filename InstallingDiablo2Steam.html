Diablo II Resurrected - Steps for Installing the Steam Version

The purpose of this page is to show the required steps for installing the Diablo II Resurrected game, as you'll need the game to be installed before you can install the mod Reimagined which is covered in a different guide.  

First you will need to create a Battle.net account (free) with a login/password with them.  You will be purchasing the Diablo II Resurrected through the Steam platform in a bit later in this guide.

 

Step 1: Download the Steam App for Windows 10/11

Steam is used to play or purchase games on the Steam platform.

  A separate Battle.net>Steam connection will be covered soon where Steam's version will use your Battle.net login/password automatically.

Go here to download the Steam Application:
Store.steampowered.com/about
 

After installing the Steam App you will use your Battlenet login/password & this will allow access to the games that you have purchased and allow for free game updates.

 

The Step 1 above is for the USA region, which uses the Global installer that should work for most countries.

Europe & Korea: Same location, the store will automatically detect your region and present prices in EU or Korean Won.

China: The official website linked above often works, but community features may be blocked without a VPN.  You can access the official Chinese version with a limited game library at store.steamchina.com

 

Step 2: Install the Steam Application

Install the Steam App & create a Steam login/password if you have not done so already. 
 

Important: You can only launch Diablo II Resurrected on Steam if you purchased the game on Steam!

Step 3: Connecting your Battle.net & Steam accounts

This is done so that the game can use your battle.net login & password automatically anytime you're playing with the Steam version.

Go to battle.net & click on your account, then click on “Connections” on the left as seen in the picture below.  

Alternatively you can go here: account.battle.net/connections

Find “Steam” and click "+Connect" as seen in the red circle in the picture below.

 

 Warning: A new steam account cannot be linked for one year after disconnecting from Battle.net. (disconnect and reconnect 365 day cooldown period).  Reconnecting the same original Steam account does not trigger this cooldown, allowing for easier switching back & forth.

Only one Steam account can be linked to a Battle.net account at a time.

 

Click “Continue” to go to the Steam website and follow the instructions.

 

 

If successful the Battle.net account page from earlier shows this, where it says it's connected to your Steam username & profile.   Now any purchased games on Steam will use your Battle.net information automatically.

 

 

When using the Steam version of Diablo II Resurrected, the Battle.net Desktop App is no longer needed!  Also Steam is forced open anytime you play Diablo II Resurrected, even when using a shortcut pointing to D2R.exe.

 

Step 4: Install the game in Steam

Everything on this page is optional.  It's recommended to uncheck both boxes in the red circle.  It's best to choose updates on the game when you're updating the Reimagined mod & not have them apply automatically.  The Desktop Shortcut this creates also points to a launcher file and not D2R.exe and should be avoided too.

If you wish to install the game in a custom location click “Change Folder”

If you wish to change the text and speech languages click “Change Languages”

Click on “Start Install” to begin the download and install process.

 


 

Step 5: Installs the game & is ready to Play!

The blue button on the bottom left changes depending on the status of the installation process.

It will change to Play when the game has been fully installed.

 

Step 6: Close the Battle.net Desktop App from your tray icon (optional) before playing mods

Right click the Blue circle icon in your system tray & click Exit.

This is optional, but playing either the base game or with mods such as Reimagined can be started by manually creating a shortcut on your desktop, start menu or taskbar pointing to D2R.exe, which bypasses the need for having the Battle.net Desktop App running in the background.  More importantly, launching directly from such shortcuts pointing to D2R.exe will not trigger automatic Diablo II Resurrected game patches/hotfixes by their developers.  

Sometimes these patches can break offline mod features or content, so it's best to update the game when you're updating the Reimagined mod OR every 30days with the DRM message.  Every thirty days, you must open the Battle.net Desktop App & click the blue Play button where the game (without mods running) opens and clears the message for another 30 days.  Otherwise before the character select screen a message will remind you that this must be performed.  If at that time there's an update, the Battle.net Desktop App will require the game be updated before launch at that time.

 

The Reimagined mod is always designed to work with the current version of Diablo II Resurrected.  Around the same time you update the Reimagined mod, it's a good idea to open the Battle.net Desktop App once to check for updates.

 

Optional: Customizing the Battle.net Diablo II Resurrected Settings

Click the white wheel next to Play & click “Game Settings”.

 

Summary of Battle.net Desktop App Settings & Diablo II Resurrected Settings:

Can change if Battle.net Launches when you start your PC

Change what happens on Game Launch (keeping Battle.net open or closing, etc)

The App's Language + Diablo II's Installed languages & even the text or speech languages can all be changed.

 

Can change the default Installation Directory for games.  For example, if you change this to C:\ then the next time you reinstall the game it will default to:

C:\Diablo II Resurrected\

 

Most importantly you can change how "Automatic Updates" are applied.

You can have it automatically apply them or never automatically apply.  If you choose never (recommended), the blue button says "Update" which lets you choose to push that to update the D2R.exe & related files to the current version.  Sometimes game hotfixes can break a mod's stash or introduce temporary instability preventing gameplay until the mod gets a new version update.

 

 

 

Below if you click on the “Game Settings” on the far left, then click on the “Diablo II: Resurrected” or “Expand All” to see more options.

It is NOT recommended to check the “Additional Command Line Arguments” box to launch mods because:

  1. The 30day DRM can't be cleared by hitting the “Play” button if you make that launch mods.  As the DRM requires the vanilla game without mods to launch once every 30days.
  2. If you use this, then you're having Battle.net check for game updates every time you launch.  If a troublesome hotfix is released, it might take time before Reimagined is able to fix offline changes the patch changed.  Things like stash breaking & Identifying item crash events occured due to rolling hotfix patches that could've been avoided by not opening the app.
  3. Game loads slightly faster if you skip using the Battle.net Desktop App & instead use a shortcut pointing to D2R.exe & add your additional Command Line Arguments there.
