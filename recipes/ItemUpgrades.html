Item Upgrade System

All weapons, armor and jewelry can be upgraded with various additional stats a certain number of times.

2H Weapons can be upgraded 10 times.

1H Weapons, Shields and Chest Armor can be upgraded 5 times.

Helms, Gloves, Belts and Boots can be upgraded 3 times.

Jewelry can be upgraded 2 times.

Items still respect their data size limits (e.g. +All Skills cannot go past 7). 

To find Stat Caps cross-reference the itemstatcost.txt entry with a stat calculator

e.g. https://docs.google.com/spreadsheets/d/1vOA1tJoTLBr3oIVIsJhZ0SW18GAvPTK3i24Ca6cJivQ/copy

When upgrading set items you must remove all other items from that set from your character first. 

Put related set items in shared stash. Save and Exit the game, join a new game, and then upgrade your intended set item. 

Rinse and repeat for any other set pieces.

Failing to do so will apply the upgrade to the set bonus and will only be effective when set bonuses are active.
