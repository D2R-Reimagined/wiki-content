Installation Guides

Below is a compilation of useful information and guides.

File Locations

D2R Mod Files: C:\Program Files (x86)\Diablo II Resurrected\mods\Reimagined

Saved Game Files: C:\Users\%USERPROFILE%\Saved Games\Diablo II Resurrected\mods\Reimagined

Your install and save locations may vary depending on your setup!

We aren't IT support, find your install and saved game directories if it differs from the above (you may have installed D2R on a different drive; you use D2RMM, etc.)

Making Backups (Manual Steps)

In File Explorer, navigate to your D2R install and Saved Games directories.

Right click the mod folder and Reimagined folder, select ‘compress to…’ and choose ZIP file. Rename the ZIP file to something memorable, and move it somewhere easy to remember (e.g. your desktop).

Remember to periodically create new ZIP folders so that you don't lose progress in case of crashes, exceeding your save file data limit, etc..
 

Reimagined Desktop Launcher (BETA)

Introducing the Reimagined Desktop Launcher (RDL) - a simple way to install, update, tweak and run the Reimagined mod!

RDL is a precursor solution to the official Reimagined Launcher, which is still in development.

The Reimagined Desktop Launcher was developed by Ultimate Supporter Slamina.

Any feedback, issues or suggestions, please speak to Vinge.

D2RMM Users: RDL makes changes to mod files in your D2R install directory (above) - you will need to copy the ‘advanced’ and ‘data’ folders to D2RMM\mods\Reimagined for them to work.

Download the Reimagined Desktop Launcher

Download Link: /installguides/reimagined_desktop_launcher_v6 (beta).zip

Download and unzip reimagined_desktop_launcher_v6 (beta).zip onto your Desktop.

The RDL only works when launched from your Desktop - this makes it easy to find, and simple to launch from.

The launcher will find your D2R Install directory, wherever it is. All you have to do is make sure the RDL folder is on your Desktop.

  • In the RDL folder you'll see a number of files and folders, and Install Instructions.
  • The RDL includes executables that are used by files to perform their various functions. These can be found in the Advanced\One-Time Installs\config directory. Nothing is installed on your PC.

If anything isn't working, refer to the Installation Troubleshooting section below.
 

Installing & Updating Reimagined

These steps assume you have already downloaded and unpacked the Reimagined Desktop Launcher (see above)

Your Save Game files are unaffected by these steps

Follow these steps to install and/or update Reimagined to the latest version:

1. Download the latest version of Reimagined from Nexus Mods - this can take a few minutes.

2. Move the downloaded .7z file from your Downloads folder to the Reimagined Desktop Launcher folder.

  • Recommended: delete any previous .7z files from this folder (if any).

3. Double-click ‘Install Reimagined (Install-Update)’.

  • If you see a warning press ‘More Info’ and click ‘Run Anyway’.
  • You should see a black command window open - let the process finish.
    • If you see an error, or the window closes quickly, refer to the troubleshooting section below.

4. The mod is now installed/updated and ready to play!

If you wish to install Advanced Tweaks, read on below.
 

Play & Backup

To play the mod, double-click either of the ‘Play Reimagined’ files. This replaces the shortcut way of launching the mod!

  • ‘Play Reimagined (Daily Backup)’ - Launches the game and creates a backup of your Reimagined Saved Game files with today's date. If a backup exists from today, it will overwrite it.
  • ‘Play Reimagined (Session Backup)’ - Launches the game and creates a backup of your Reimagined Saved Game files each time you launch the game, including the date/time.

Backup files are located under your Saved Games folder, in a folder called Reimagined Automatic Saved Game Backups - see File Locations above.
 

Advanced Tweaks

This section assumes you have complete the RDL Installation steps above.

In addition to installing and updating the mod, you can use the RDL to install pre-packaged changes.

Once you have run Install Reimagined, navigate to the Advanced folder - here you should see a number of files.

To install changes, simply double-click the relevant file(s).

To uninstall changes simply re-run ‘Install Reimagined (Install-Update)’.

As mentioned above, whenever you run ‘Install Reimagined (Install-Update)’ any manual or advanced tweaks changes will need to be redone.

If anything isn't working, refer to the Installation Troubleshooting section below.
 

Expanded Shared Stash Tabs

Description: Adds 4 additional Shared Stash tabs.

Installed this change before? Re-run ‘Expanded Stash Mod Directory Install'.

First time installing? 

Clear all items from your Shared Stash tabs - move them to personal stash, inventory, or to a mule.

Run: 

  1. Expanded Stash Mod Directory Install
  2. Expanded Stash Saved Game Install (this overwrites your existing Shared Stash file)

 

Forced Terror Zones

Description: Permanently terrorizes a pre-selected list of popular areas. Zones will no longer cycle every hour.

Run: ‘Forced Terror Zones Install’

Areas terrorized:

  • Abandoned Tower (Countess)
  • Catacombs (Andariel)
  • Arcane Sanctuary (Summoner)
  • Durance of Hate (Mephisto)
  • Chaos Sanctuary (Diablo)
  • Nihlathak's Temple (Nihlathak)
  • Throne of Destruction
  • Worldstone Chamber (Baal)
  • Moo Moo Farm (Cows)
     

Remove Splash Charm Dust Effect

Description: Removes the dust particle effect when using the Splash Charm (Collin's Charm).

Run: ‘Splash Charm Effect Graphic Removal Install’


Two Skill Points / Level

Description: Characters gain 2 skill points per level instead of 1.

Run: ‘Two Skills Per Level Install’

  • This change does not work retroactively.
  • Skill points already earned cannot be modified.
  • If you uninstall this change characters will return to earning 1 skill point per level.
  • Using a Token of Absolution to reset your character will refund all skill points earned up to that point (whether 1 or 2 points).
     

CASC Fastloading

Description: Unpacks all game files - reduces load times, increases performance and FPS, client response times are better, etc.

NOTE: Requires an extra 41GB of disk space to unpack

Run: 

  1. Step 1. CASC Fastloading (takes 5-10 minutes to complete)
  2. Step 2. CASC Fastloading
  3. Run Install Reimagined (Install-Update)
    • Note: this will uninstall other advanced tweaks, re-install them as desired afterwards

 

To uninstall this change:

  1. Navigate to your Reimagined mods folder: \Diablo II Resurrected\mods\Reimagined\Reimagined.mpq\
  2. Delete the data folder.
  3. Re-run Install Reimagined (Install-Update).
  4. Optional: empty your recycle bin to free-up the disk space again.

 

Installation Troubleshooting

Below is a list of possible failure points you may encounter during RDL installation:


Error: Cannot find 7zr.exe / Path Too Long
  • Problem: You didn't extract to the desktop.
  • Resolution: Double-check where you extracted to, move the folder to your desktop if necessary.

Error: Reimagined isn't loading!

Problem: Reimagined 7z file not in RDL folder.

Resolution: Ensure you have downloaded the latest Reimagined 7z file from Nexus and moved it to the Reimagined Desktop Launcher folder.

  • Re-run Install Reimagined (Install-Update) to load the mod, and try again


Error: Advanced Tweaks aren't working!

Problem: Advanced folder isn't in the Reimagined mod folder. You likely didn't run Install Reimagined first.

Resolution: If your advanced folder is missing, here are a few things to try:

1. Run Install Reimagined (Install-Update).

  • Once done, try re-applying your advanced tweak changes

2. If still not working:

  • Delete your RDL folder
  • Re-download/extract to your desktop
  • Run Install Reimagined
  • Try installing advanced tweaks


D2RMM

Using a Mod Manager allows you to easily install a variety of mods together (assuming they play nice) without having to know anything about modding.

Reimagined is packed full of expansive features and QOL updates, so installing extra mods isn't always necessary. 

Check the list of Reimagined features below to see if a 3rd-party mod is still necessary.

 

When using D2RMM your active mod files will be now be located under the D2RMM\mods\ folder instead of Diablo II Resurrected\mods\.

Any changes you make to Reimagined files needs to be done in the D2RMM mods directory.

If using the RDL (above), you will need to copy the file changes over from Diablo II Resurrected\mods\Reimagined\Reimagined.mpq\ to D2RMM\mods\Reimagined\.
 

File Steps

1. Download D2RMM from Nexus Mods.

2. Open the D2RMM folder - navigate to the mods directory.

  • You should see a text file called 'PUT YOUR MODS HERE' - leave this window open, we'll use it later.

3. Open a new File Explorer window and navigate to your Reimagined mod folder under your D2R Install directory.

4. Copy the Reimagined.mpq folder (this should contain a Data folder and ModInfo.json).

5. Paste the Reimagined.mpq folder into the D2RMM/mods folder from step 2.

  • We recommend renaming the folder to Reimagined

D2RMM Steps

6. Open D2RMM as Administrator.

7. Once open, click the three dots in the bottom-right corner and select 'Refresh Mod List'.

8. Reimagined should appear in the mod list - click the box on the left-side to enable it.

The Reimagined folder must always be first in the list of mods. Click and drag the line to the top if it isn't.

9. Navigate to the Settings tab (at the top).

  • 9a. Click on General.
    • Change 'Output Mod Name' to 'Reimagined'.
    • Toggle 'Automatically install mods before running the game'.
  • 9b. Click on Game Launcher.
    • Under 'Extra Game Args', paste the following: -mod Reimagined -txt
    • Feel free to add any other args you want after everything else.

10. Navigate back to the Mods tab.

11. Click Install Mods.
 

Play time!

12. Hit 'Run D2R'.
 

Loot Filters

This guide assumes you have already downloaded and installed D2RMM (see above)

If you find Quilted Armor with no graphics, leave them on the ground. Do not attempt to identify or sell them.

Reimagined doesn't yet include a built-in Loot Filter.

Below we'll take you through a step-by-step guide on how to install and configure a loot filter. We recommend Pseudo Loot Filter.
 

File Steps

1. Download Pseudo Loot Filter from Nexus Mods.

  • Select ‘Mod manager download’ to automatically download and load the mod into D2RMM.
  • Select ‘Manual download’ to download the mod to your Downloads folder - extract the mod into your D2RMM/mods folder.
     

D2RMM Steps

2. In D2RMM, click the 3 dots in the bottom-right, and select ‘Refresh Mod List’.

3. You should now see Pseudo Loot Filter appear in the list of mods - click the box to the left to enable the mod.

4. To configure the loot filter, click ‘settings’ to the right.

5. All items are enabled by default (the slider is on the right-side) - clicking the slider DISABLES the item (this affects all rarities, white, magic, rare, set, unique).

  • Disabling an item simply means that when a disabled item would drop, the game chooses to drop nothing instead.
  • Disabling items does not affect drop chances of other items.
  • It may take a few runs to tweak this list to your liking.

6. Once you've finished disabling undesirable items, click out of the menu.

7. Click ‘Install Mods’ in the bottom-right.

8. Hit ‘Run D2R’.


D2RMM FAQ:

My mod isn't showing up in the mod list on D2RMM?

  • You didn't put the mod in D2RMM/mods, you didn't unzip the file you downloaded, or it's in a folder that D2RMM doesn't understand
  • Run D2RMM as administrator

Reimagined Features:

If you have questions about whether something is included in Reimagined, or would like to see something added, DM one of the Devs, or post in the suggestions-and-feedback channel of our discord channel.

A number of Mod Creators gave express permission to include their mod in Reimagined. Where this is the case credit is given on the Reimagined Nexus Mods page.

The below (non-exhaustive) list of mods are already functionally covered in Reimagined in some form:

Inventory

  • Expanded Inventory / Stash / Cube

Items

  • Increase Drop Rates (for all items, runes, gems, etc.)
  • Sunder Charms (dropped by Bosses, super uniques, etc.)
  • Melee AOE Small Charm (Collin's Charm)
  • Perfect Gem Drops / Short Gem Names (there is only one type of Gem in Reimagined)

Systems

  • Item Upgrades
  • Item Dyes
  • Horadric Forging (various cube recipes allow rerolling, crafting, etc.)
  • Corruptions (implemented via Orb of Corruption)
  • Socket Punching
  • Stackable Runes & Gems (runes drop as stacks; gems are collected in a bag)
  • Rune Easy Upgrade / Downgrade (tooltip printed on rune stacks)

Mercenaries

  • Expanded Merc Equipment (all equip slots useable for mercs)

QOL

  • Show Item Level
  • Show Item Quality (all items contain [N], [X], [E] tags to show quality)
  • Numbered Rune Names
  • Better Rune Visibility (runes already named, numbered and colored for easier visual identification)
  • Light Pillars (used on various important items)
  • Short Potion Names (renamed already to HP1, MP1, etc.)
  • Town Cast
  • Token of Absolution (sold by Potion Vendors)
  • Short Scroll Names (renamed to TP and ID)
  • Move Cain and Teleports (characters always start in center of town, Cain is there too)
  • Breakpoints and Recipes panels (already included in various forms in-game)
  • Terror Zones (TZs span entire acts for an hour already, no need to pick them)

Monsters

  • Monster Density (density custom-set in every area already)

Environment / UI

  • Diablood
  • Skill Icons (all skills have been colorised and textured for each class, including Mercs)
