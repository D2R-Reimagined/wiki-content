How it works

Easy as 1 2 3!

 

There are two bat files & a freeware command line version of 7zip included.  This tool never needs to be redownloaded.  Read the included readme file for more information.  It's purpose is to simply make uncompressing future reimagined downloads even easier.  No longer need any installed software to uncompress the Reimagined mod.

 

After unzipping this, there is a new folder created called “Download Reimagined Updates Here” inside the game installation.  You simply need to run bat file 1 which deletes all other .7z files inside this folder only.  Next you'll need to download the Reimagined .7z off Nexus and save that inside this folder.  Finally doubleclick the bat file 3 & the reimagined mod will be uncompressed and overwrite your old installation.  Your saved games are located in a different location and should remain safe when updating the mod.

 

Now the mod is installed!  

Easy!

 

 

Bat files are fully transparent & editable by right clicking and choosing edit with notepad.  A bat file is simply Microsoft Windows Command Prompt executing commands one after another.  Feel free to change the path to better suit your own needs if your game was installed elsewhere.
