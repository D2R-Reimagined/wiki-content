Advanced Tweaks for Reimagined 2.04

Advanced tweaks like these are from the efforts of various gamers & would not have been possible without the community on Discord.  Please join the growing team of gamers & let us reimagine the future together!

IMPORTANT: Because most of the options below either adjust a single file or alter many files, when a new version of Reimagined gets installed you will have to reapply every tweak afterwards.  Keep this in mind when deciding which of these are of interest.  You could use them all together if you wanted, they are all optional.

It is recommended to first experience the mod by itself as default before trying any of these advanced changes.

 

Typically until now the gamer was required to either edit or merge multiple folders manually, opening the door for user errors.  Allowing the batch files to automate the overwrite should make turning on or off a modification extremely easy for gamers.  Every .bat file is fully transparent, you are able to right-click and choose edit.  Notepad shows the lines of commands & Microsoft Command Prompt simply runs them as written.

The install bat file simply grabs one file and replaces the file used inside the real final directory the game uses.  So double click the install to tryout the change, and double click uninstall to go back to the default.

Short description of each advanced tweak

MOST TWEAKS BELOW CAN BE INSTALLED OR UNINSTALLED BY SIMPLY RUNNING THE .BAT FILE.

Expanded Stash

This increases the shared stash tabs from 3 to 7.  Every additional tab is able to hold 2.5M gold as well allowing for an additional 10 million more gold to be stored.  More freedom with more storage.

 

Forced Terror Zones

This forces the following very popular zones to always be terrorized!  While changing your computer system's clock can extend throne of destruction or cow level tz sessions, instead this will keep these popular areas terrorized permanently makes your life easier.

Forced terrorized zones: Countess, Summoner, Nilathak, Andy, Meph, CS, Throne of Destruction, Baal, Cows

 

Potion & scroll removal from most monster drops

With potions and scrolls no longer dropping from most mobs, missing important gear becomes less of an issue.  They can still be purchased from the vendor.

 

Remove the Splash Charm's effect graphic

By default when using the Splash Charm, a red swoosh effect is present letting the gamer know how large the area for the effect is covering.  You can turn off this graphic, while the damage will continue to hit mobs making it easier to see things while fighting.

 

Expanded Aspect Ratio limit (xx% zoomed out)

This increases the viewable area on-screen, where Blizzard has allowed upto 19:10 ratio to be used in the game.  This allows players to teleport horizontally faster than a typical 16:9 ratio.  Currently the steps listed are for Nvidia video card users only & works on 1080p and 4k monitors too.

 

Much Faster Loading game

This reduces the amount of time it takes for both the game's initial loading and also reduces the loading screens when changing in-between acts/zones.  This has been reported to remove microstutters & slightly reduce the minimum system requirements (except for the hard drive game storage size requirement).

Expanded Aspect Ratio limit (xx% zoomed out)

This increases the viewable area on-screen, where Blizzard has allowed upto 19:10 ratio to be used in the game.  This allows players to teleport horizontally much faster than a typical 16:9 ratio.  Currently the steps listed are for Nvidia video card users only, works on 1080p & 4k monitors too.

When playing on a non 19:10 display, this will introduce black bars on the top and bottom of the screen resulting in a little less visible area.  However, the game will be zooming out to compensate, allowing for more loot to be displayed on screen at once.  Allows you to see monsters further on the edge of the screen & extends how far left and right you are able to teleport making runs go even faster.

Only writing the steps for Nvidia Graphics card users for now.  Will update for AMD users this summer.  Note that Windows needs to be operating at this aspect ratio while playing the game in fullscreen to be successful.  So limited visibility while alt+tabbing will remain until you quit the game and change back to your typical desktop resolution.  Suggest trying this at least once, especially if playing on a larger display.
