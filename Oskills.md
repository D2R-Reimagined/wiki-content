# Oskills & CTC Only Skills

## Oskills
OSkills (+X to Skill) only provides +3 to the class that owns the skill, total. This limit of +3 to owned skill includes multiple sources of the same OSkill. All other classes receive the full benefit.

If a +X to Skill also contains (Class Only) then only that class benefits from the skill level increase and gains the full effect.
