Pandemonium Keys

Pandemonium Keys form part of the event chain for the ‘Pandemonium Event’.

Pandemonium Keys can be obtained from Key Wardens:

  • The Countess (Act 1, Forgotten Tower) - Key of Terror
  • The Summoner (Act 2, Arcane Sanctuary) - Key of Hate
  • Nihlathak (Act 5, Halls of Vaught) - Key of Destruction

Pandemonium Keys can also be obtained as a random drop from Hell Terror Zone enemies.

See this page for more information.

Key Storage: Keychain

The keychain is used to store PandemoniumKeys.

Acquiring

You can purchase a keychain from any vendor.

How it works

The keychain uses a "credit" system. Each key that is added to the keychain adds one equivalent credit.

Key credits are displayed separately.

Storing Keys

Transmute a Pandemonium key with the keychain in the Horadric Cube to add a key to the Keychain. You can add up to three keys of the same type to the keychain at a time.

The Keychain will then show the available credits for each key.

Retrieving Keys

Purchase a key grabber from any vendor - there is a key grabber for each Pandemonium Key.

Transmuting the keychain with a grabber extracts a key from the keychain and subtracts a credit from the keychain.
