Gems

All gems now have a single quality level. Vanilla D2R gem qualities (e.g. chipped, flawed, normal, flawless, perfect) have been removed from regular drop tables.

Some hardcoded drop tables (Andariel, Hell Forge, Gem Shrines) will still drop inferior gem types. 

Transmute inferior gems with the Gem Bag in the Horadric Cube for credits!

All gems drop from monsters that are level 18 and up (roughly Act 2, Arcane Sanctuary), and require level 18 to use.
 

Gem Storage: The Gem Bag

All new characters start with a Gem Bag in their inventory. 

Replacements can be purchased from vendors should you somehow misplace it.

How it works

The Gem Bag is used to store large numbers of gems, and uses a "credit" system. Each gem added to the bag adds one credit. 

Credits are universal allowing you to retrieve any of the seven gem types as needed.

The Gem Bag can hold up to 65,000 gems.

Storing Gems

Add the Gem Bag and up to 99 assorted gems to the Horadric Cube and press transmute to add them. 

The game will crash if you attempt to add more than 99 gems to the bag at once.

You can add up to 3 inferior gems (chipped, flawed, normal, flawless, perfect) of the same type to the bag at once. Each adds 1 credit.

Retrieving Gems

Gems can be retrieved using ‘grabbers’ (also purchased from vendors).

Transmute the Gem Bag with an appropriate Gem Grabber to retrieve a single gem.

You can also transmute a grabber by itself to cycle through the types.
